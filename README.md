# 美容師国試クイズ

美容師国家試験の過去問をクイズ形式で出題するWebアプリです。

https://flatring.github.io/biyoushi-quiz/

## 収録データ

現在は**第53回（2025年）**のみ収録しています。

- 全**55問**を収録
- 問題データの参考元：[公益財団法人 理容師美容師試験研修センター 過去問題](https://www.rbc.or.jp/exam/past_question/)

## 機能

- **全問モード** — 55問を出題（ランダム順 / 番号順 を切り替え可能）
- **苦手問題モード** — 直近で間違えた問題だけを出題
- **ダッシュボード** — 総正答率・科目別正答率・間違えた問題一覧を表示

## 注意事項

- 回答履歴は端末のlocalStorageに保存されます。**端末をまたいでの同期はできません**
- ブラウザのデータを消去すると履歴もリセットされます
