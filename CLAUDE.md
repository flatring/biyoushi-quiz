## Git ルール

git履歴を変更・リモートに影響する操作は**自律的に実行禁止**。

- 禁止操作: `git commit` / `git push` / `git rebase` / `git reset` / `git push --force` など
- ユーザーが明示的に指示した場合のみ実行可
- その場合も毎回「〜を実行してよいですか？」と確認してから実行する

## コーディング規約

- コメントは英語
- インデント: スペース4つ
- 外部ライブラリ不使用（index.html 1ファイル完結）

## コミットメッセージ

日本語動詞始まり（例: `追加:` `修正:` `更新:`）
